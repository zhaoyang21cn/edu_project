– joinClassroomWithOption:succ:failed:

加入指定roomID的课堂

- (void)joinClassroomWithOption:(TICClassroomOption *( ^ ) ( TICClassroomOption *option ))configOption succ:(TCIVoidBlock)succ failed:(TCIErrorBlock)failed

Parameters

configOption

加入课堂配置项

Discussion

注意：TICClassroomOption 的 controlRole 和 avOption.privateMapKey 参数必填，controlRole代表进房之后使用什么规格音视频参数，参数具体值为客户在腾讯云实时音视频控制台画面设定中配置的角色名（例如：默认角色名为user, 可设置controlRole = @“user"）；avOption.privateMapKey 相当于房间秘钥，1.1.0 版本后必传，详细见 https://cloud.tencent.com/document/product/680/17891#2.6-.E8.AF.BE.E5.A0.82.E7.AE.A1.E7.90.86
