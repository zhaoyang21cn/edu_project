– onRecvMessage:

收到消息

- (void)onRecvMessage:(TIMMessage *)msg

Parameters

msg

消息对象

Discussion

上面四个消息回调只会在收到特定的消息类型时才会回调，而该方法收到所有的消息都会回调（可能和上面的回调方法重复），包括文本消息、自定义消息、表情、图片等；如果开发者只用到了文本和自定义消息，只监听上面四个回调即可，如果还需要用到更多类型的消息，则需监听该回调方法，自行解析（该方法抛出的是IMSDK中的TIMMessage对象）
